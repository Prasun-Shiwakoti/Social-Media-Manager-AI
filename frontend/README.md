# Social Media Manager AI

A modern, premium frontend for a Social Media Management platform powered by AI.

## Features

- **Dashboard**: Real-time stats and quick actions.
- **AI Content Creator**: Generate images and captions.
- **Scheduler**: Plan and schedule posts with a calendar view.
- **Analytics**: Detailed performance metrics with charts.
- **Social Inbox**: Manage comments and DMs.
- **Insights**: AI-driven trend analysis.

## Tech Stack

- React + Vite
- TailwindCSS
- shadcn/ui
- Recharts
- Lucide React
- React Router v6

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Project Structure

- `/src/components`: Reusable UI components.
- `/src/pages`: Application pages.
- `/src/services`: API service placeholders.
- `/src/store`: State management (Zustand).
- `/src/lib`: Utilities.

## Customization

- **Theme**: Colors are defined in `src/index.css` using CSS variables.
- **Components**: Built with shadcn/ui, located in `src/components/ui`.
